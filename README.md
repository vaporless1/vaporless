Vaporless — минималистичное приложение для трекинга парения вейпа.

Приложение предназначено для пользователей, которые хотят отслеживать личные показатели и формировать осознанный подход к своему поведению. Vaporless не содержит лишних функций и сосредоточено на простоте и удобстве использования.

Возможности vaporless:
 -Учет действий с помощью удобного счетчика
 -Наглядное отображение прогресса
 -Минималистичный интерфейс без перегруженных элементов
 -Быстрый доступ ко всем функциям
 -Работа без подключения к интернету
-Серия дней без курения вейпа

Приложение не требует регистрации, не собирает и не передает персональные данные, вся информация хранится локально на устройстве пользователя.

Vaporless не является медицинским приложением и не предоставляет рекомендаций. Приложение предназначено исключительно для личного учета и самоконтроля.

Подходит для пользователей, которые ценят простоту, аккуратный дизайн и удобные инструменты для отслеживания привычек.
